# CRUD de Gestión de Proyectos

Sistema web desarrollado en PHP y MySQL para la administración y seguimiento de proyectos.

## Funcionalidades

- Registrar proyectos
- Editar información de proyectos
- Eliminar proyectos
- Listar y buscar proyectos
- Gestión de estados de proyectos
- Autenticación de usuarios
- Reportes

## Tecnologías Utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Git y GitHub

## Instalación

1. Clonar el repositorio.
2. Importar la base de datos en MySQL.
3. Configurar las credenciales de conexión.
4. Ejecutar el proyecto en XAMPP.
